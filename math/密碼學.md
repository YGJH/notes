### 古典密碼 vs 現代密碼
- 古典密碼ex: 凱薩密碼、維吉尼亞密碼etc. 主要就是把訊息轉換掉然後讓敵人就算竊取走密文也很難懂
- 現代密碼學ex: RSA, ECC（橢圓曲線密碼學）　因為自從有了計算機這神奇的東西後，古典密碼幾乎成了笑話，任何的古典密碼＂幾乎＂都可以被電腦硬破，或者＂統計學＂找出來，所以現代密碼學更傾向加密每一個byte

----

- 古典密碼學主要關注資訊的保密書寫和傳遞，以及與其相對應的破譯方法
- 現代密碼學不只關注資訊保密問題，還同時涉及資訊完整性驗證( Data integrity)（訊息驗證碼）、資訊發布的不可抵賴性( Non-repudiation)（數位簽章）、以及在分散式計算中產生的來源於內部和外部的攻擊的所有資訊安全問題

- <a href="https://aikawa0617.medium.com/%E5%AF%86%E7%A2%BC%E5%AD%B8-%E4%B8%80-%E5%8F%A4%E5%85%B8%E5%AF%86%E7%A2%BC%E5%AD%B8-455e43038241">來源</a>

### KSA
```python=
def KSA(key):
    S = list(range(256))
    j = 0
    for i in range(256):
        j = (j + S[i] + key[i % len(key)]) % 256 #移動S[i]跟S[j]
        S[i], S[j] = S[j], S[i] #交換S[i]跟S[j]
    return S

```

----


### PRGA
```python=
i = 0
j = 0
def PRGA(input_byte)):
    global i,j,S
    i = (i + 1) mod 256
    j = (j + S[i]) mod 256
    S[i], S[j] = S[j], S[i] #Swap(s[i] , s[j])
    return input_byte ^ S[(S[i]+S[j]) % 256]
```

def egcd(a, b):
    if a==0:
        return (b,0,1)
    else:
        g, x, y = egcd(b % a , a)
        return (g, y-(b//a) * x , x)
def modinv(b, n):
    g, x, _ = egcd(b , n)
    if g==1:
        return x % n

### fermat_factorization
```python=
def fermat_factorization(N) :
    """
    Fermat's factorization for close p and q
    link : https://en.wikipedia.org/wiki/Fermat%27s_factorization_method
    """
    a = nroot(N,2)[0]
    b2 = a*a - N
    b = nroot(N,2)[0]
    count = 0
    while b*b != b2:
        a = a + 1
        b2 = a*a - N
        b = nroot(b2,2)[0]
        count += 1
    p=a+b
    q=a-b
    return p, q
```

### nth_root
<a href="https://riptutorial.com/python/example/8751/computing-large-integer-roots">來源</a>
```python=
def nth_root(x, n):
    # Start with some reasonable bounds around the nth root.
    upper_bound = 1
    while upper_bound ** n <= x:
        upper_bound *= 2
    lower_bound = upper_bound // 2
    # Keep searching for a better result as long as the bounds make sense.
    while lower_bound < upper_bound:
        mid = (lower_bound + upper_bound) // 2
        mid_nth = mid ** n
        if lower_bound < mid and mid_nth < x:
            lower_bound = mid
        elif upper_bound > mid and mid_nth > x:
            upper_bound = mid
        else:
            # Found perfect nth root.
            return mid
    return mid + 1
```

### 尤拉定理
- 尤拉定理，由費馬小定理的推廣
- 任何互質的數a, n([a,n] = 1), 則滿足$a^{\phi(n)} = 1\ mod\ n$
- 換言之，若a, n互質，則指數將會被關入$\phi(n)$的循環中
<!-- ![](https://i.imgur.com/B9sWHQA.png) -->
- $若n為質數，則\phi(n)=n-1$
- 反正就是不用一定要質數，只要a,n互質就好

----

- $a^{\phi(n)} = 1\ mod\ n$
<!-- ![](https://i.imgur.com/KLJvCG7.png) -->
- 若n為質數，$\phi(n) = n - 1$

若n為合數，$n = p^a * q^b * r^c$
$\phi(n) = (p-1)p^{a-1} * (q-1)q^{b-1}*(r-1)r^{c-1}$
<!-- ![](https://i.imgur.com/Ew0CJWE.png) -->

----

### RSA
- 要怎麼做?
1. 要找出兩個超大的質數p, q
2. N = p*q
3. e = 任意質數
4. 找到$phi = (p-1)*(q-1)必須跟e互質$
5. $d = inverse(e ,phi) -> ed mod=1
6. 把p, q丟掉

## RSA 漏洞
- 一些容易被分解的N
    - 有小質數組成
    - 有兩組以上的n共用質數p, q
    - 由太相近的兩質數組成
    - 組成N的質數p，(p-1)是個k-smooth數
<a href = "http://factordb.com/">factordb</a>

----

- 費瑪因式分解
- 如果合數n剛好是由相近的兩質數組成
- 可以在$\sqrt(n)$內將其分解
- $N = p*q = (x-a)*(x+a) = x^2*a^2$
- a要很小

## 波拉德Pollard's p-1

且K不大，是個可以走完的數

## ECC
- 橢圓曲線密碼學，是一種基於橢圓曲線數學的公開密鑰加密演算法
- 函式為:$y^2\ =\ x^3\ +\ ax\ +\ b$根據參數的不同會有不同形狀

## ECC
- 剛剛定義的<font color="#f00">P + Q + R = 0</font>
- 帶入交換得:P + Q = -R

## ECC
- 假設曲線為$y^{2}\ =\ x^{3}-7x+10$
- 曲兩點P、Q: (3,4),(1,2)
- 兩點的直線為:$y=x+1$
- 求得R = (-3,-2)
- $P + Q + R = 0 => P + Q = (-3,2)$
<img src='https://hackmd.io/_uploads/S1GIDSTaR.png' style="height: 200px">

## ECC
- 在這基礎之上加上mod的概念
- $y^{2}\ =\ -x^{3}+7x+10\ (mod\ 19)$ 
- 剛才的P，Q = (1,2),(3,4)
- $R(-3,\ -2)由於mod\ 19=>R(16,17)$
- $P\ +\ Q\ =-R\ = (16,2)$

## 具體用法
- 通常ECC不用於直接加密，而是通過ECDH算出共享密鑰
```
Encrypt: (rG, M+rP)
Decrypt: (M+rP) - k(rG) = M+r(kG)+k(rG) => M
```
P: 公鑰
G: 基點
r: 隨機數
M: 要加密的訊息
