- 封包內容就像是![[Pasted image 20250419211801.png]]
- 所以直接上程式碼
```python
import requests
import os
import subprocess
# 要下載的 .ts 檔案網址清單
base_url = "https://wbug.xyz/videos/8e05d6ac6d7f876533d4e57aceb20e10/play"
ts_urls = [f"{base_url}{i}.ts" for i in range(1, 217)]  # 自訂數量
# 建一個資料夾來存 .ts
os.makedirs("ts_files", exist_ok=True)
# 下載 ts 檔案
with open("file_list.txt", "w") as f:
    for idx, url in enumerate(ts_urls):
        ts_path = f"ts_files/part{idx}.ts"
        print(f"下載中: {url}")
        r = requests.get(url)
        with open(ts_path, "wb") as ts_file:
            ts_file.write(r.content)
        f.write(f"file '{ts_path}'\n")
# 合併成一個大 .ts（使用 ffmpeg 的 concat）
concat_cmd = [
    "ffmpeg", "-f", "concat", "-safe", "0",
    "-i", "file_list.txt", "-c", "copy", "output.mp4"
]
subprocess.run(concat_cmd)
print("影片組合完成！輸出檔案為 output.mp4")
```

你抓到的這串 `.ts` 檔案其實是影片串流的一部分，格式應該是 **HLS (HTTP Live Streaming)**。`.ts` 就是 MPEG-TS（Transport Stream）格式，是一種用來傳輸影音資料的容器格式，常被串流影片服務用來分段播放影片。

---

### 你看到的那些亂碼其實是：

- **二進位影音資料**：像是音訊、視訊幀、字幕等，會直接以壓縮格式封裝起來（通常是 H.264 + AAC）。
    
- **FFmpeg 標記**：你看到裡面出現了 `FFmpeg`、`Service01` 等字樣，這是常見的影片轉碼工具，表示影片可能是經過它轉檔處理的。
    
- **加密/混淆過的內容**：部分串流服務會加密 `.ts` 檔案，除非你有正確的金鑰或用對播放器，不然用一般文字編輯器或抓包工具打開就是亂碼。
    

---

### 那這個封包可以怎麼用？

1. **播放器（如 VLC）可以播放**：你把 `.ts` 存下來用 VLC 開應該能正常播。
    
2. **用 FFmpeg 合併 + 解碼**： 如果你有一連串 `.ts` 檔案，你可以用：
    
    ```bash
    ffmpeg -i "concat:file1.ts|file2.ts|file3.ts" -c copy output.mp4
    ```
    
    來合併這些片段變成完整影片。
    
3. **若有 `.m3u8`**：這是 HLS 的索引檔，會列出所有 `.ts` 片段，直接丟到 VLC 或 ffmpeg 也能播放/下載完整影片。
    

---

### 如果你是在分析串流網站怎麼傳資料：

- `.ts` 是影片段落（通常幾秒一個）
    
- `.m3u8` 是播放清單
    
- 有些網站會加密 `.ts`（你得抓到金鑰才能解開）
    

---

用 `.m3u8` 的話超方便的，基本上一行 `ffmpeg` 指令就可以搞定整部影片。

---

### 🔍 m3u8 是什麼？

`.m3u8` 是一種播放清單格式，裡面會列出一堆 `.ts` 檔案的網址，也可能還有不同畫質版本。HLS 就是靠這個來串流的。

---

### ✅ 工具需求

你只需要：

- 安裝好 [`ffmpeg`](https://ffmpeg.org/download.html)
    
- 有 `.m3u8` 的網址或檔案
    

---

### 📥 用 ffmpeg 下載 m3u8 串流並轉成 mp4

```bash
ffmpeg -i "https://example.com/video/playlist.m3u8" -c copy -bsf:a aac_adtstoasc output.mp4
```

#### 📌 說明：

- `-i`: 輸入的 m3u8 網址
    
- `-c copy`: 表示直接把影片資料複製下來，不重新編碼（速度快）
    
- `-bsf:a aac_adtstoasc`: 把音訊格式從 ADTS 轉成更通用的格式
    
- `output.mp4`: 輸出的檔案名稱
    

---

### 🌐 如果網址擋住你（403、要 cookie、token 等）

你可以加一些 header，例如：

```bash
ffmpeg -headers "User-Agent: Mozilla/5.0\r\nReferer: https://example.com\r\n" \
-i "https://example.com/video/playlist.m3u8" -c copy -bsf:a aac_adtstoasc output.mp4
```

---


代 cookies.txt的方法 （比較通用）
```

ffmpeg -headers "User-Agent: Mozilla/5.0" \
       -cookies "$(cat cookies.txt | tr '\n' '; ')" \
       -i "https://example.m3u8" \
       -c copy output.mp4

```

### 📦 進階功能：下載字幕、特定畫質？

m3u8 裡面可能有多個畫質版本，想選的話要先看清單內容：

```bash
ffmpeg -i "https://example.com/master.m3u8"
```

它會列出有哪些 variant，然後你再手動選一個 `.m3u8` 網址來用。
